# Quick Start Guide

## Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Demo Flow

### Client Portal
1. Go to http://localhost:3000
2. Click "Client Portal"
3. Navigate between three tabs: Intake, Mandatory Disclosures, Additional Documents
4. Upload documents in "Mandatory Disclosures" tab
5. Click "?" icon next to any document for help
6. Ask chatbot: "Is this necessary?"
7. Watch progress bar update

### Lawyer Portal - Snapshot
1. Click "Lawyer Portal" from home
2. Default view shows Snapshot section
3. See pie chart of document collection
4. View uploaded vs pending statistics
5. Click "Send Reminder to Client"

### Lawyer Portal - Documents
1. Click "Documents" tab
2. View all uploaded documents
3. Scroll to "Organize Documents"
4. Click "Organize Files"
5. Watch 3-second animation
6. See organized folder structure

### Lawyer Portal - Forms
1. Click "Forms" tab
2. Click "Open Form Editor"
3. Click "Auto-Populate from Documents"
4. Watch fields fill with data
5. Click "View" on any green source button
6. See document viewer with highlighted text
7. Click "Return to Form"
8. Click "Refresh Preview"
9. See actual PDF with your data
10. Click "Download PDF"

### Lawyer Portal - AI Assistant
1. Click "AI Assistant" tab
2. Ask: "What is the September 2024 balance of the client's Chase account?"
3. Get answer with citation
4. Try other financial questions
